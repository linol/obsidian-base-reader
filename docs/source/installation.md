# Installation

## Install from PyPI (future)

```bash
pip install obsbase
```

## Development install

If you work on the project locally:

```bash
git clone https://github.com/your-name/obsidian-base-reader
cd obsidian-base-reader
pip install -e .
```

## Optional dependencies

For DataFrame support:

```bash
pip install pandas
```

For documentation build:

```bash
pip install -e ".[docs]"
```
